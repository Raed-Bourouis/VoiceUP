# VoiceUP : Online Chat App

This is project is for academic purposes.

An android app is available in the [releases section](https://github.com/Raed-Bourouis/VoiceUP/releases/tag/v1.0.0-release)

 **Alternatively,**
to build the app, clone this repo, then run the following commands:

```bash
> cd voiceup
> flutter pub get
> flutter run [-d <device_id>]
```
you may omit the `-d <device_id>` flag to use the default connected device

Or run
 ```bash
 > flutter devices
 ```
 for a full list of connected devices

## NOTE!!

A supabase url and key are required for this project, save them in a .env file as `SUPABASE_URL` and `SUPABASE_ANON_KEY`
